Recursive binary search

Recursive binary search is an algorithmic approach for searching for an element within a sorted array or list. It follows the divide-and-conquer strategy, repeatedly dividing the search space in half until the desired element is found or determined to be absent.

Here's a high-level overview of how recursive binary search works:

  1. Base case: The algorithm starts with a sorted array and defines a base case. If the array is empty or the search space has been narrowed down to a single element, the algorithm either returns the index of the element if it matches the search key or indicates that the element is not found.
  2. Divide: The algorithm finds the middle element of the current search space (array) by calculating the average of the lower and upper indices. If the middle element matches the search key, the algorithm returns the index.
  3. Conquer: If the middle element is greater than the search key, the algorithm recursively performs a binary search on the lower half of the array. Otherwise, if the middle element is less than the search key, the algorithm recursively performs a binary search on the upper half of the array.
  4. Combine: The result of the recursive calls is propagated back up the call stack until the final result is obtained.
