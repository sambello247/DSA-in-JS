Binary Search

Binary search is an efficient algorithm used to search for a specific target value within a sorted list or array. It follows a divide-and-conquer approach by repeatedly dividing the search space in half until the target value is found or determined to be absent

Here's how the binary search algorithm works:

  1. Start with a sorted list or array of elements.
  2. Define two pointers, low and high, which represent the indices marking the current search range.
  3. Calculate the middle index as (low + high) / 2, rounding down if necessary.
  4. Compare the target value with the element at the middle index.
    • If they match, the target value is found, and the search terminates successfully.
    • If the target value is less than the middle element, update high to be one index less than the middle index. This restricts the search space to the lower half of the list.
    • If the target value is greater than the middle element, update low to be one index more than the middle index. This restricts the search space to the upper half of the list.
    • If the target value is greater than the middle element, update low to be one index more than the middle index. This restricts the search space to the upper half of the list.
    • If the search range is empty and the target value hasn't been found, the search terminates unsuccessfully.

The key advantage of binary search is that it dramatically reduces the search space with each iteration, as it eliminates half of the remaining elements at every step. This leads to an efficient search algorithm with a time complexity of O(log n), where n is the number of elements in the sorted list.

Binary search is most effective when applied to sorted lists or arrays since it relies on the ability to divide the search space in half. If the list is not sorted, the binary search algorithm won't work correctly, and a different search algorithm may be more appropriate.
