Bubble Sort

Bubble sort is a simple sorting algorithm that works by repeatedly comparing adjacent elements and swapping them if they are in the wrong order. Bubble sort is not very efficient, but it is easy to understand and implement.

Bubble sort example

This is how bubble sort work:

[-6, 20, 8, -2, 4]
[-6, 20, 8, -2, 4] -> [-6, 8, 20, -2, 4] Swap since 20 > 8
[-6, 8, 20, -2, 4] -> [-6, 8, -2, 20, 4] Swap since 20 > -2
[-6, 8, -2, 20, 4] -> [-6, 8, -2, 4, 20] Swap since 20 > 4

End of array. Elements swapped? Yes? Repeat the comparision.

[-6, 8, -2, 4, 20]
[-6, 8, -2, 4, 20] -> [-6, -2, 8, 4, 20] Swap since 8 > -2
[-6, -2, 8, 4, 20] -> [-6, -2, 4, 8, 20] Swap since 8 > 4
[-6, -2, 4, 8, 20]

End of array. Elements swapped? Yes? Repeat the comparision.

This continues until non of the element can't be swapped, which means the array has been completely sorted
